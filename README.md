# OEVM_laba1
оценка характеристик ПК

## Задание.
Написать bash-скрипт, который выводит на экран характеристики ПК в
следующем формате.

Дата

Имя учетной записи

Доменное имя ПК

Процессор:
- Модель 
- Архитектура 
- Тактовая частота 
- Количество ядер 
- Количество потоков на одно ядро 

Оперативная память:
- Всего 
- Доступно 

Жесткий диск:
- Всего 
- Доступно 
- Смонтировано в корневую директорию /
- SWAP всего 
- SWAP доступно 

Сетевые интерфейсы:
- Количество сетевых интерфейсов
